# Network Connectivity Tool

A lightweight Python utility designed to check remote host availability and measure latency using the `ping3` library. This project utilizes an isolated virtual environment to manage dependencies securely.

## Prerequisites

Before running this utility, ensure you have Python 3.7 or higher installed on your system.
